# primeiroestudodecaso

Estudo de caso da versão anterior do projeto **WPM Gestão Interna**.

## Finalidade
Este repositório preserva o primeiro estudo de caso publicado do projeto, com abordagem mais visual e narrativa, útil para explicar:
- origem do problema
- contexto humano e operacional
- motivação da solução
- visão da primeira versão consolidada

## Status
Este material permanece ativo como referência histórica da **V33**.

## Observação importante
Este não é o case principal mais atualizado do projeto.

O repositório oficial do estudo de caso atualizado é:
- `WPM-RECEP-CASE`

## Relação com os demais repositórios
- `WPM-SISTEMA-RECEP` → sistema principal oficial
- `WPM-DOCUMENTACAO-RECEP` → documentação oficial
- `WPM-RECEP-CASE` → case oficial atualizado
- `primeiroestudodecaso` → case anterior, mantido por valor narrativo e visual
